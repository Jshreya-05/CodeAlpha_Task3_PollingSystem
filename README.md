# CodeAlpha_Task3_PollingSystem
# 🗳️ CodeAlpha Internship - Task 3

## 🧾 Project: Polling Smart Contract

This project demonstrates a decentralized polling system built using Solidity.

### 🚀 Features:

* Create a poll with multiple options
* Vote securely (one vote per user)
* Prevent double voting
* Time-based voting system
* Retrieve winner after poll ends

### 🛠️ Tools Used:

* Solidity
* Remix IDE
